# 📖 About This Repository

This repository contains a collection of **C++ programming practice programs** created while learning the fundamentals of programming.

These programs cover the most important beginner concepts including:

- 🔢 Arithmetic Operations
- 🔁 Loops
- 🔀 Conditional Statements
- 🧮 Mathematical Calculations
- 📊 Number Problems
- 📚 Basic Algorithms

Each program is written in **simple and beginner-friendly C++ code**, making it easy to understand and practice.

---

# 📂 Programs Included

| Program | Description |
|---------|-------------|
| 🧮 Calculator | Performs basic arithmetic calculations |
| 🔢 Armstrong Number | Checks whether a number is Armstrong or not |
| ➕ Combination | Calculates combinations (nCr) |
| 💰 Compound Interest | Calculates compound interest |
| 💯 Display Numbers | Displays numbers up to 100 |
| ⚖️ Even or Odd | Checks whether a number is even or odd |
| 🔁 Even/Odd Integers | Prints even and odd integers |
| 🌱 Fibonacci Series | Generates Fibonacci sequence |
| ✖️ Factorial | Finds factorial of a number |
| 📈 Largest Number | Finds the largest among three numbers |
| 📏 Data Type Size | Displays size of int, float, double and char |
| ➗ Fraction | Performs fraction calculations |
| 👋 Hello World | First C++ program |
| ➕➖ Loop Operations | Demonstrates loop operations |
